E-Commerce Web Application

A full-stack e-commerce web application built with React (Vite) and Node.js, featuring a product catalog, shopping cart, and a back-end REST API. Users can browse products, add them to a cart, and place orders, while the back end serves product and order data.

Features


Product catalog with product images and details
Shopping cart (add, update quantity, remove items)
Order placement handled through the REST API
Product and order data served by a Node.js back end
Responsive layout for desktop and mobile


Tech Stack


Frontend: React (Vite), JavaScript, HTML, CSS
Backend: Node.js, Express (REST API)
Data storage: JSON files (file-based data in backend/data)


Project Structure

/frontend        # React (Vite) client
   /public       # images and static assets
   /src          # components and pages
   index.html
   vite.config.js
/backend         # Node.js + Express API
   /data         # JSON data files
   /routes       # API routes
   server.js

Getting Started

Prerequisites


Node.js and npm installed


Backend

bashcd backend
npm install
npm start

Frontend

bashcd frontend
npm install
npm run dev

The frontend runs on the local Vite dev server (shown in the terminal, e.g. http://localhost:5173).
